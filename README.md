# Web Application Security Assessment

## Target
testphp.vulnweb.com (Intentionally Vulnerable Application)

## Author
Rushikesh Pawar  
Date: 07-01-2026

## Objective
To perform a web application security assessment using
black box testing techniques and identify common vulnerabilities.

## Tools Used
- Kali Linux
- Nmap
- Burp Suite (Community Edition)
- Firefox Browser

## Testing Type
Black Box Testing

## Vulnerabilities Identified
- SQL Injection (High Risk)
- Cross-Site Scripting (XSS) (Medium Risk)
- Sensitive Information Disclosure (Low Risk)

## Repository Structure
- Report → Complete security assessment report
- Vulnerabilities → Individual vulnerability details
- Risk → Risk summary
- Screenshots → Testing evidence

## Disclaimer
This project was conducted strictly for educational purposes
on a legally vulnerable test application.
